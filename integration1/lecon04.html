CSS

CSS (Cascading Style Sheets) est un langage qui contrôle l’apparence des pages web.

Séparer la structure de la forme

  • HTML définit la structure et le contenu.
  • CSS définit la présentation : couleurs, polices, marges, disposition.

Depuis son introduction en 1996, CSS a beaucoup évolué. Aujourd’hui, on parle de CSS3, qui regroupe de nombreux modules modernes (flexbox, grid, animations, variables, etc.) largement utilisés par les développeurs web.

Un style est formé de règles appliquées à des éléments HTML.

Le flux Internet

De la saisie d'une adresse dans votre navigateur jusqu'à la réception du document demandé, voici une description brève des étapes d'une requête effectuée d'un ordinateur client (votre ordinateur) vers un serveur.

Étapes d'une requête effectuée d'un ordinateur client vers un serveur

Côté client

Côté serveur

1. Saisie d'une adresse www dans le navigateur.
2. Le navigateur contacte le serveur HTTPS situé à cette adresse.
3. Le serveur HTTPS reçoit la requête du navigateur.
4. Le serveur HTTPS recherche le document Web.
5. Le serveur HTTPS envoie le document Web.
6. Votre navigateur reçoit le document.
7. Votre navigateur traite le code source.
8. Le navigateur affiche la pag Web.

À quel moment les CSS interviennent-ils ?

C'est à partir de la dernière étape de la liste précédente qu'interviennent les CSS.

  1. Si le document en contient, ils décrivent au navigateur l'aspect visuel que doit prendre le document HTML.
  2. Si le navigateur comprend et reconnaît les CSS, il convertit la page en un document que vous pouvez visualiser et avec lequel vous pouvez interagir.
  3. S'il n'en comprend qu'une partie, il ignore ce qu'il ne peut déchiffrer.
  4. S'il ne comprend pas du tout les CSS, il affiche une version brute du document HTML.

Rédiger des CSS

La méthode la plus simple pour rédiger les CSS consiste à utiliser un éditeur de texte brut tel que Visual Studio Code. Les fichiers CSS sont en effet de simples documents texte.

Il existe trois façons d’ajouter du CSS à une page HTML :

  1. En ligne (inline)
  2. Interne (balise <style>)
  3. Externe (fichier .css)

1. En ligne (inline)

Les règles de style en ligne peuvent être rédigées à même le balisage HTML, directement dans l’attribut style d'une balise. La règle s’applique uniquement à cet élément.

Exemple de CSS en ligne:

                        <p style="color: red;"> ... </p>

Règle générale, cette pratique est à éviter. Mais il existe des situations particulières où les styles en ligne sont pertinents.

2. Interne (balise <style>)

Dans l’en-tête du document HTML (<head>), à l’intérieur d’une balise <style>.

Les règles définies dans <style> s’appliquent à tous les éléments de la page qui correspondent aux sélecteurs.

Exemple de feuille de style interne

                    <!DOCTYPE html>
                    <html lang="fr">
                      <head>
                        <style>
                        p {color: red;}
                        </style>
                      </head>
                      <body>
                      (...)

Dans l'exemple ci-dessus, le navigateur affichera toutes les balises <p> de la page dans la couleur red.

3. Externe (fichier .css)

Dans un fichier séparé, lié à la page par une balise <link>.

C’est la méthode recommandée. Elle permet de réutiliser les styles et de séparer clairement le contenu (HTML) de la présentation (CSS).

Exemple de liaison d'un fichier externe dans un document HTML

                    <!DOCTYPE html>
                    <html lang="fr">
                      <head>
                        <link rel="stylesheet" href="styles.css">
                      </head>
                      <body>
                      (...)

Exemple de contenu du fichier styles.css

                        @charset "utf-8";
                    p {
                      color: red;
                    }

Dans l'exemple ci-dessus, toutes les pages liées au fichier styles.css appliqueront les règles de styles qui y sont inscrites.

@charset "utf-8";

Il ne doit y avoir ni caractère, ni commentaire, ni espace avant la déclaration du codage de caractères au début d'une feuille de style externe. Pour plus de détails, consultez les règles du W3C .

La déclaration du codage est utilisée pour assurer l'interprétation correcte de certaines valeurs d'attribut qui peuvent contenir des caractères spéciaux.

Les composantes d'une feuille de style

Une règle CSS est composée de deux parties principales :

  1. Le sélecteur
    • Indique quel élément HTML sera stylisé.
    • Exemple : p cible tous les paragraphes.
  2. La déclaration
    • Définie entre { }.
    • Contient une ou plusieurs propriétés et valeurs.
    • Exemple :
      p {
                                  color: red;     /* propriété: color, valeur: red */
                                  font-size: 1.4em; /* propriété: font-size, valeur: 1.4em */
                                }

Résumé visuel

                    sélecteur {
                      propriété: valeur;
                      propriété: valeur;
                    }

Bonnes pratiques pour les règles de style

La méthode à utiliser pour bien organiser le code et faciliter sa consultation :

  • indenter chaque ligne (avec la touche )
  • faire un saut de ligne après chaque déclaration

Exemple d'un code lisible et bien organisé

                    h1 {
                      font-family: Georgia, "Times New Roman", serif;
                      font-size: 1.4em;
                      color: #0033cc;
                    }

Règle de style écrite dans retour à la ligne. Moins facile à lire.

                        h1 {font-family:Georgia, "Times New Roman", serif;font-size:1.4em;color:#0033cc;}

Sélecteur

Sélecteur = le ou les éléments HTML auxquels une règle CSS s'applique.

Le sélecteur de l'exemple précédent est h1. Il s'agit, dans ce cas, d'un élément HTML simple.

  • Il est écrit avant la première accolade ({).
  • Notez qu'il est écrit sans les chevrons (<>) qui encadrent la balise correspondante dans le document HTML.

Sélecteur groupé

Un sélecteur groupé applique la même règle à plusieurs éléments, séparés par des virgules.

h1, h2, h3 { ...

Dans l'exemple ci-dessus, la règle de style va s'appliquer à toutes les balises <h1>, <h2> et <h3>.

Sélecteur composé

Un sélecteur composé cible des éléments en relation hiérarchique (séparés par des espaces, sans virgules).

ul li a { ...

Dans l'exemple ci-dessus, la règle s’appliquera uniquement aux <a> contenus dans un <li> placé dans un <ul>.

Déclaration

Les déclarations sont placées entre les accolades pour les isoler des sélecteurs. Chaque déclaration est composée d'une propriété et d'une valeur CSS séparées par un deux-points (:), le tout se terminant par un point-virgule (;)

                    sélecteur {
                      propriété: valeur;
                      propriété: valeur;
                    }

Propriété

Une propriété CSS décrit un aspect visuel d’un élément HTML (ex. couleur, taille, police).

Il existe plus de 300 propriétés en CSS3, et ce nombre augmente avec les évolutions du langage.

Chaque propriété est suivie d’une valeur, séparée par un deux-points (:).

Exemples de propriété :

  • font-family : police de caractère
  • font-size : taille de la police
  • color : couleur du texte

Liste non exhaustive de propriétés CSS

Valeur

Une valeur précise l’apparence d’une propriété CSS.

Elle peut être : nom de couleur, code de couleur, unité de longueur, pourcentage, nombre, etc.

Exemples de valeur :

  • Georgia, "Times New Roman", serif
  • 1.4em
  • #0033cc

Notes sur la couleur en CSS

La couleur est un aspect fondamental de la présentation visuelle : elle affecte tant le texte que les fonds, bordures et autres éléments graphiques. En CSS, on dispose de plusieurs façons de définir une couleur, chacune adaptée à des besoins différents (précision, compatibilité, effets de transparence…).

Méthode Description Exemple d’utilisation
Notation hexadécimale longue Code commençant par #, suivi de 6 chiffres hexadécimaux (RR, GG, BB). color: #0a4dee
Notation hexadécimale courte Un seul chiffre par composante, dupliqué implicitement. color: #f06;
RGB Fonction rgb(r, g, b) avec valeurs 0–255. color: rgb(50, 10, 204);
RGBA Comme RGB mais avec canal alpha (opacité 0–1). color: rgba(50, 10, 204, 0.4);
HSL Fonction hsl(h, s%, l%) : teinte 0–360°, saturation %, luminosité %. color: hsl(100, 50%, 28%);
HSLA HSL avec canal alpha. color: hsla(24, 0%, 98%, 0.6);
Couleurs nommées Mots-clés prédéfinis (≈140 en CSS). color: olive;

Les sélecteurs du DOM

Un sélecteur CSS sert à cibler un ou plusieurs éléments du document HTML afin d’appliquer un style.

Il existe différents types de sélecteurs, chacun adapté à une situation particulière :

  • Sélecteurs de base
  • Sélecteurs combinés
  • Sélecteurs d’attributs
  • Sélecteurs pseudo-classes

Sélecteurs de base

Sélecteur Description Exemple d’utilisation
Type (balise) Cible tous les éléments d’un type donné. h2 {}
Sélectionne tous les éléments <h2>
Classe (.nom) Cible les éléments ayant un attribut class. .important { font-weight: bold; }
Identifiant (#id) Cible un élément unique avec un attribut id. #header { background: gray; }
Universel (*) Cible tous les éléments du document. * { margin: 0; }

Sélecteurs combinés

Sélecteur Description Exemple d’utilisation
Descendant (E F) Cible un élément contenu dans un autre. li a {}
Sélectionne tous les élément <a> descendants de <li>.
Enfant direct (E > F) Cible uniquement les enfants immédiats. li > a {}
Sélectionne tous les élément <a> qui sont des enfants d'un éléments <li>, mais sans les autres éléments <a> de la page
Frère adjacent (E + F) Cible l’élément immédiatement suivant. h1 + p {}
Sélectionne chaque élément <p> qui vient immédiatement après un élément frère <h1>, sans les éléments <p> suivants
Frères généraux (E ~ F) Cible tous les frères suivants. h1 ~ p {}
Sélectionne tous les éléments <p> qui sont précédés d'un élément frère <h1>

Sélecteurs d’attributs

Sélecteur Description Exemple d’utilisation
[attr] Cible les éléments possédant un attribut. [title] { text-decoration: underline; }
[attr="valeur"] Cible un attribut ayant une valeur précise. input[type="text"] { border: 1px solid #ccc; }
[attr^="val"] Valeur commençant par… a[href^="https"] { color: green; }
[attr$="val"] Valeur finissant par… a[href$=".pdf"] { color: red; }
[attr*="val"] Valeur contenant… a[href*="exemple"] { color: orange; }

Sélecteurs pseudo-classes

Sélecteur Description Exemple d’utilisation
:hover État au survol. a:hover { text-decoration: underline; }
:first-child Cible le premier enfant. p:first-child {}
Sélectionne tous les éléments <p> qui sont des premiers enfants de leur parent
:last-child Cible le dernier enfant. p:last-child {}
Sélectionne tous les éléments <p> qui sont des derniers enfants de leur parent
:nth-child(n) Cible l’enfant à une position donnée. p:nth-child(3) {}
Sélectionne tous les éléments <p> qui sont des troisièmes enfants de leur parent
:not() Exclut un sélecteur. p:not(.important) { color: gray; }
::first-letter Sélecteur de la première lettre d'un élément p::first-letter {}
Sélectionne la première lettre de l'élément <p>
::first-line Sélecteur de la première ligne d'un élément p::first-line {}
Sélectionne la première ligne de l'élément <p>
