BEM – La classe modificateur pour le lien de la page active

Rappel : l'annotation BEM pour nommer les classes

Vous devrez dorénavant utiliser l'annotation BEM pour nommer les classes dans tous vos travaux, y compris le TP examen.

Revoir les notes de la leçon 7 : L'annotation BEM

La classe de modificateur (voir la méthode BEM pour la définition) peut être utilisée pour définir le lien de la page courante.

Il suffit d'ajouter cette classe (ex.: menu__lien--actif) à la classe de base du lien (menu__lien) et de définir la mise en forme du lien actif à l'aide du sélecteur de classe.

On peut insérer deux classes consécutive en laissant un espace entre elles, comme ceci: 

<a class="menu__lien menu__lien--actif">...

L'indicateur de lien externe

Les bonnes pratiques recommandent d’indiquer visuellement les liens menant vers un autre site. On peut, par exemple, ajouter une petite icône à droite du lien au moyen d’une classe et de la propriété background-image.

Pensez aussi à ajouter l’attribut target="_blank" pour ouvrir ces liens dans un nouvel onglet.

Optimiser un média pour le Web

Objectif de l’optimisation :
Réduire le poids du fichier pour accélérer le chargement des pages Web, améliorer l’expérience utilisateur et limiter la consommation de bande passante et d’énergie.

Voici les étapes pour optimiser un média pour le Web:

  1. Ouvrir l’image dans Photoshop.
  2. Aller dans Image > Taille de l’image et ajuster aux dimensions souhaitées.
  3. Choisir Fichier > Exportation > Enregistrer pour le Web (hérité).
  4. Sélectionner le format approprié : JPEG pour les photos, PNG-24 pour les images avec transparence
  5. Ajuster le niveau de qualité (entre 60 % et 80 % pour les JPEG).
  6. Cocher Optimisé pour réduire le poids du fichier.
  7. Vérifier la taille du fichier dans l’aperçu.
  8. Enregistrer avec un nom de fichier descriptif, sans accents ni espaces, et ajouter le suffixe indiquant la largeur à la fin (ex. _w600 pour 600 px de large).
    Exemple : article-titre-entete_w600.jpg

Leçon 11: TP examen (25%)

Le TP examen est un travail pratique synthèse sur l'ensemble des connaissances acquises jusqu'ici dans le cours.

Contrairement à un TP régulier, je ne peux vous suggérer des solutions complètes: vous devrez vous débrouiller par vous-mêmes. Si vous êtes dans un cul de sac, je peux vous mettre sur une piste.

Les notes de cours sont permises, ainsi que tous les travaux réalisés depuis le début de la session. Il est donc recommandé de réviser toutes les notes de cours et les travaux pour vous préparer adéquatement.

Exercice révision

Objectif

Réviser les notions de structure HTML, de mise en forme CSS, d’optimisation d’images et de mise en page réactive.

Consignes

  1. Créer un dossier de travail comprenant une page nommée index.html et une feuille de style styles.css.
  2. Lier la feuille de style à la page HTML.
  3. Intégrer :
    • Une navigation d'entête vers des pages fictives.
    • Un titre principal (h1) fictif.
    • Au moins deux sections contenant chacune : un sous-titre (h2), un ou deux paragraphes de texte fictif (générateur de Lorem Ipsum ) et une image optimisée dans Photoshop.
  4. Utiliser la balise picture et l’attribut srcset pour rendre les images réactives selon la largeur de l’écran.
  5. Sur les écrans larges, faire flotter chaque image à droite du texte, avec une marge interne pour l’espacement.
  6. Ajouter un court formulaire sous le contenu principal contenant : un champ texte pour inscrire son nom et un bouton intitulé « S’abonner à l’infolettre ».
  7. Applique des styles cohérents : typographie, couleurs, marges, largeur de contenu maximale et alignements.
  8. Teste la page sur différents formats d’écran pour vérifier la réactivité.

Leçons 12 à 15: TP Long (40%)

Dans le cadre de ce travail pratique, vous réaliserez un site Web complet à plusieurs pages sur la thématique de votre choix : c’est ce qu’on appelle un webzine.

Un webzine est un magazine publié en ligne, conçu autour d’un thème précis qui peut être très varié : musique, voyage, sport, cuisine, environnement, sciences, etc.

Vous créerez votre webzine de A à Z, du concept initial jusqu’à la mise en ligne. Vous pouvez dès maintenant commencer à réfléchir à la thématique que vous souhaitez explorer.
